# 📊 Data Professional Survey Breakdown Dashboard (Power BI)

## 📌 Overview  
This project presents an **interactive Data Professional Survey Dashboard** built using **Power BI** to analyze responses from data professionals across the globe.

The dashboard provides insights into **demographics, salaries, programming preferences, career challenges, and job satisfaction**, helping understand trends in the data industry.

---

## 🚀 Key Features  

### 🌍 Demographic Insights
- Country-wise distribution of survey respondents  
- Highlights global participation trends  

### 🧠 Career Entry Analysis
- Difficulty level of breaking into the data field  
- Categorized into:
  - Easy  
  - Difficult  
  - Very Difficult  
  - Very Easy  
  - Neutral  

### 💻 Programming Preferences
- Most popular programming languages among data professionals  
- Python dominates as the top choice  

### 💰 Salary Insights
- Average salary by job role:
  - Data Scientist  
  - Data Engineer  
  - Data Analyst  
  - Data Architect  
  - Database Developer  
- Helps compare compensation across roles  

### 😊 Job Satisfaction Metrics
- Work-life balance rating  
- Salary satisfaction rating  

### 📊 Key Metrics (KPIs)
- Total Survey Respondents: **630**  
- Average Age: **29.87 years**  
- Work-Life Balance Score: **5.74 / 10**  
- Salary Satisfaction Score: **4.27 / 10**
